\# Dataset



This project uses the \*\*New Plant Diseases Dataset (Augmented)\*\*, which is derived from the PlantVillage dataset.



\## Dataset Information



\- Number of Classes: 38

\- Total Images: Approximately 87,000

\- Image Size: 224 × 224 pixels

\- Format: RGB JPG Images



\## Download



The dataset can be downloaded from Kaggle:



https://www.kaggle.com/datasets/vipoooool/new-plant-diseases-dataset



\## Dataset Structure



Train/

Valid/



Each class contains images of healthy and diseased plant leaves.



\## Note



The dataset is \*\*not included\*\* in this repository because of its large size and Kaggle licensing. Please download it using the link above and place it in the appropriate directory before training the model.
